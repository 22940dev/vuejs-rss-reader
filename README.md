# Vue RSS Reader
A simple client side only RSS Reader written with Vue.js
![Screenshot](https://i.irq.pw/u/10.58.59-09.04.21.png)


## Features
- Customizable feeds
- Theming
- Importing/Exporting of feeds for sharing


---

```
npm install
```
